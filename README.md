# LTTC-QuantumDynamics-Project
Fortran 90 implementation of quantum wave packet propagation using the split-operator method. Built as a graded assignment for Prof. Arjan Berger's Quantum Dynamics module at LTTC Winter School January 2025 (TCCM Master's). Extends tutorial code with harmonic oscillator eigenstates, Hermite polynomials, and superposition analysis.
